# Описание

Данное web-приложение автоматизирует процесс разработки раскладки в походах и не только

# Зависимости

python 3.8

npm 8.4.1

# Быстрый старт:

Запуск клиентской части на react

1. Для запуска клиентской части, перейти в папку `cd frontend`
2. Установить зависимости `npm install`
3. Запустить react приложение `npm start`

Запуск серверной части, python django app

1. Для запуска серверной части перейти в папку `cd v-pohod`
2. Установить записимости `pip install - r requirements.txt`
3. Настроить базу данных `python manage.py migrate`
4. Создать админскую учетку: `python manage.py creatsuperuser`. Задать ему логин и пароль.
5. Запутить серверную часть: `python manage.py runserver`

Открыть страницу *http://localhost:3000/*
